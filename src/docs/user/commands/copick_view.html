Command: copick view

Usage:
copick view  mode

The copick view command switches the main viewport between the three Copick views. The mode argument is one of:

  • 3d – the ChimeraX 3D graphics canvas, where tomograms and annotations are shown.
  • gallery – the thumbnail gallery for browsing runs.
  • details – the details view showing metadata for the selected run.

These are the command equivalents of the floating 3D (🧊), details (ℹ) and gallery (📸) buttons in the Copick tool. Opening a run's tomogram (via copick open run, the run tree, or the gallery) automatically switches to the 3D canvas.

Examples:

copick view 3d
copick view gallery
copick view details

See also: copick open run, copick start.
