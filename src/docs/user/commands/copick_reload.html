Command: copick reload

Usage:
copick reload

The copick reload command reloads the current project from its configuration file, refreshing the run tree and the picks, meshes and segmentations tables. Pending pick edits are stored before reloading. This is the command equivalent of the Reload button in the Copick tool.

Example:

copick reload

See also: copick start.
