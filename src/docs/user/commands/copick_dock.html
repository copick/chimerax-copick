Command: copick dock

Usage:
copick dock  tool-name  [ side ] [ tabWith other-tool ]

The copick dock command repositions any open ChimeraX tool window, addressed by the tool's name. It is handy for scripting a reproducible window layout (for example, a “demo” arrangement). It works on any tool, not just Copick, and does not require a copick session to be started.

tool-name is matched against the names of currently open tools (case-insensitive, exact match preferred, otherwise a prefix match). Names containing spaces must be quoted, e.g. "Command Line Interface".

The window is placed according to side, one of:

  • left, right, top, bottom – dock to that edge of the main window.
  • float – detach the window so it floats freely.

Alternatively, the tabWith keyword tabs the window together with another tool's window (the window is placed in that tool's current dock area). When tabWith is given it takes precedence and any side value is ignored. Either a side or tabWith must be supplied.

Examples:

copick dock "Log" bottom
copick dock Copick left
copick dock "Model Panel" float
copick dock "Log" right tabWith Copick

See also: copick start, ui tool.
