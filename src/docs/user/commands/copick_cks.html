Command: cks

Usage:
cks [ shortcut ]

The cks (copick keyboard shortcuts) command enables copick keyboard shortcuts. After it runs, keys typed in the graphics window are interpreted as copick shortcuts (for example showing/hiding particles, stepping through picks, and switching visualization modes).

With no argument, cks switches the graphics window into shortcut-input mode. With a shortcut argument, that shortcut is executed directly.

Example:

cks
Copick / chimerax-copick
