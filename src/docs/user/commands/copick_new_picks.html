Command: copick new picks

Usage:
copick new picks object-name [ user_id id ] [ session_id id ]

The copick new picks command creates a new, empty set of picks for the pickable object object-name in the active run, shows it, and switches the right mouse button to the mark plane picking mode so points can be placed immediately. This is the command equivalent of the New button on the Picks tab.

optionmeaning
user_id User ID for the new picks. Defaults to the project's user ID (or ArtiaX if none is set).
session_id Session ID for the new picks. Defaults to the next available manual-N identifier in the run.

object-name must be a pickable object defined in the project configuration (manage these via Edit Object Types). A run must be open first (see copick open run).

Examples:

copick new picks ribosome
copick new picks ribosome user_id alice session_id manual-2

See also: copick open / show / hide, Copick URIs.
