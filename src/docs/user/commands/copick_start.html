Command: copick start

Usage:
copick start config-file

The copick start command opens the Copick tool (if it is not already open) and loads the project described by the copick configuration file config-file. Running it again with a different file switches the session to that project.

Example:

copick start /path/to/copick_config.json

See also: copick new, copick reload.
