Command: copick new

Usage:
copick new config-file [ config_type filesystem | portal ] [ dataset_ids id,id,... ] [ root_dir path ] [ name text ] [ description text ]

The copick new command creates a new copick configuration file at config-file and loads it into the Copick tool.

optionmeaning
config_type Type of project to create: filesystem (default) for a local overlay, or portal for a CryoET Data Portal project.
dataset_ids Comma-separated dataset IDs (required for portal), e.g. dataset_ids 10301.
root_dir Root directory for a filesystem project's data (defaults to a folder next to the config file).
nameProject name.
descriptionProject description.

Examples:

copick new /tmp/project.json root_dir /tmp/project_data
copick new /tmp/portal.json config_type portal dataset_ids 10301

See also: copick start.
