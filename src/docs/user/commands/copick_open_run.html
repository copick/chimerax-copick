Command: copick open run

Usage:
copick open run run-name [ tomo_type type ] [ zarr_level 0 | 1 | 2 ]

The copick open run command opens a tomogram of the run named run-name in the Copick tool, making that run the active run. This is the command equivalent of double-clicking a tomogram in the run tree.

optionmeaning
tomo_type Type of tomogram to open (e.g. wbp, denoised). When given, the matching tomogram at the largest (most downsampled) voxel spacing is chosen. When omitted, a default tomogram is selected automatically.
zarr_level Resolution pyramid level to display initially: 0 = full, 1 = 2×, 2 = 4× downsampled. Defaults to the value configured in the table settings. Other levels are still available via the volume step controls after loading.

Examples:

copick open run TS_001
copick open run TS_001 tomo_type wbp zarr_level 1

See also: copick open / show / hide, copick reload.
