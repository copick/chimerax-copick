Tool: Copick

The Copick tool provides collaborative annotation of cryo-electron tomograms using the copick data model. It browses the runs of a project, loads tomograms, and shows or edits picks, meshes and segmentations, rendering them through ArtiaX.

Start the tool from the Tools → Volume Data → Copick menu, then load a project with the copick start command, or create a new one with copick new.

Layout

  • Gallery / Tree – browse runs. Double-clicking a tomogram opens it (see copick open run). A search box filters runs by name; the floating buttons switch between gallery, details and 3D views.
  • Picks / Meshes / Segmentations tabs – the entities available for the active run. Double-click a row to show/hide it; use the row buttons to create, duplicate or delete entities. The particle stepper navigates individual picks.
  • Edit Object Types – manage the project's pickable objects.
  • Reload – reload the project from its config file (see copick reload).

Scripting

Most UI actions have command equivalents so sessions can be scripted, and the corresponding command is echoed to the Log whenever you perform the action in the UI. See the Commands section and Copick URIs.

Toolbar

A Copick toolbar tab provides visualization (slab, volume, orthoplanes, surface), binning, marker/axis/surface visibility, viewing directions, and particle picking and manipulation mouse modes.
