Copick User Guide

Copick is a collaborative cryo-ET annotation plugin for ChimeraX. It uses the copick data model to browse runs, load tomograms, and view/edit picks, meshes and segmentations, rendering them through ArtiaX.

The user interface is described in the Tools section. All commands are prefixed with "copick". Picks, meshes and segmentations are addressed by their copick URI.

  •  General
    • Copick URIs  – How to address picks, meshes, segmentations and tomograms
  •  Tools
    • Copick  – Browse runs and view/annotate picks, meshes and segmentations.
  •  Commands
    • start  – start a copick session / load a config file
    • new  – create a new copick config file (filesystem or data portal)
    • open run  – open a run's tomogram
    • open, show, hide  – show or hide picks, meshes and segmentations by URI
    • new picks  – create a new empty set of picks in the active run
    • reload  – reload the current session from its config file
    • view  – switch the viewport between 3D canvas, gallery and details
    • dock  – dock a tool window to an edge, float it, or tab it with another tool
    • cks  – enable copick keyboard shortcuts
